## Breaking Changes
The following characters need to be escaped.

These will error
```shell
<
{   
<!-- Comment -->
```


When escaped, these will work
```shell
\<
\{
\<!-- Comment -->
```

... or use braces

```shell
{/* Comment */}
```

or, in some cases, expressions.

```shell
`'<'`
`'{'`
```
